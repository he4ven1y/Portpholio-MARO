## 🚀 Как запустить проект локально?
Я настроил для тебя локальный веб-сервер. Чтобы увидеть сайт прямо сейчас:
1. Запусти локальный сервер (я сделаю это в следующем шаге).
2. Открой ссылку в браузере.

## 📓 Как использовать Obsidian?
1. Открой программу **Obsidian**.
2. Выбери опцию **"Open folder as vault"** (Открыть папку как хранилище).
3. Укажи путь к этой папке: `C:\Users\Roma\.gemini\antigravity\scratch\vibe-designer-portfolio`.
4. В левой панели Obsidian ты увидишь папку `Notes/` с материалами, которые я подготовил.
